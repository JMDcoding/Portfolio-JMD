====== DOSSIER IMAGE DE FOND (BACKGROUND) ======
Placez ici votre image de fond pour toutes les pages.

Nom de fichier attendu : bg.jpg  (ou bg.png)

Le site chargera automatiquement l'image depuis ce chemin :
  assets/background/bg.jpg

RECOMMANDATIONS :
- Format : JPG (préféré pour la compression) ou PNG
- Taille recommandée : 1920x1080 px minimum
- Choisissez une image sombre ou avec une bonne profondeur pour garantir la lisibilité du texte
- Évitez les images trop lumineuses (un overlay sombre sera appliqué dessus)
